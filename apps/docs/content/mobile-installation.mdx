# Mobile Installation Guide

Install DevFleet as a mobile app without downloading from an app store. It works like a native app using PWA technology.

## iOS Installation

1. Open Safari and go to `https://app.devfleet.ai`
2. Tap the Share button
3. Select "Add to Home Screen"
4. Tap "Add" to confirm
5. Find the DevFleet icon on your home screen and tap it

## Android Installation
